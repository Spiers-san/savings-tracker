Savings Tracker
A personal finance app to track savings goals.

Features
- User login and signup
- Create savings goals with target amounts
- Add deposits to goals
- Track progress visually
- View transaction history
- Dashboard overview

Files
index.html - Login page
dashboard.html - Main dashboard
onboarding.html - Setup page
css/ - All CSS files
js/ - All JavaScript files

How to Use
1. Open index.html in browser
2. Create account or login
3. Set up initial information
4. Create savings goals
5. Add deposits to track progress

Setup
1. Create Supabase account
2. Create savings_goals and transactions tables
3. Update Supabase URL and key in HTML files
4. Open index.html to run locally
